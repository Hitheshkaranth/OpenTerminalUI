The Open-Source Financial Terminal

A professional-grade market terminal for discretionary traders, researchers, and quant teams. Real-time charting, derivatives analytics, backtesting labs, portfolio management, and 70+ technical indicators — all in one self-hosted platform.

Platform Features

Everything you need in a financial terminal — from real-time market data to institutional-grade quant workflows.

〉

Terminal-Grade Shell

Bloomberg-style GO bar (Ctrl+G), command palette (Ctrl+K), and function key shortcuts (F1-F9). Keyboard-first navigation designed for speed. Desktop and mobile-aware layouts with persistent workspace framing.

CORE
⍑

Multi-Panel Chart Workstation

Up to 9 synchronized chart panels with crosshair linking, 70+ technical indicators, drawing tools with templates, replay controls, volume profile overlay, and comparison series. Supports 1m to monthly timeframes with extended hours toggle.

CHARTING
☰

Equity Research & Security Hub

Deep security analysis with 8 tabs: overview, financials, chart, news sentiment, ownership, analyst estimates, peers, and ESG. Company fundamentals, earnings calendar, promoter holdings, and corporate actions tracking.

RESEARCH
⚙

Advanced Multi-Market Screener

Query builder with 15+ visualization modes: tables, treemaps, heatmaps, scatter plots, radar charts, box plots, bubble charts, waterfall charts, RRG quadrants, and more. Scan across NSE, BSE, NYSE, and NASDAQ with custom formulas.

SCREENER
∇

Futures & Options Analytics

Full option chain with live Greeks (Delta, Gamma, Theta, Vega, Rho), OI analysis, PCR tracking, strategy builder for multi-leg positions, IV heatmaps, expiry calendar, and futures term structure visualization.

DERIVATIVES
★

Portfolio & Risk Management

Multi-portfolio management with allocation charts, benchmark comparison, correlation heatmaps, VaR/CVaR, EWMA volatility, PCA factor exposures, stress testing, and tax lot tracking. Paper trading engine with realistic order fills and TCA.

PORTFOLIO
⚙

Backtesting & Model Lab

10+ pre-built strategy templates with intraday and daily backtesting. 3D parameter surfaces, Monte Carlo simulations, walk-forward analysis, and monthly return heatmaps. Model Lab for experiment tracking with CAGR, Sharpe, Sortino, and max drawdown metrics.

QUANT
⍟

Portfolio Lab & Strategy Blends

Multi-asset portfolio backtesting with risk parity, equal weight, and momentum weighting modes. Strategy blend definitions with up to 10 strategies and 200 assets. Rebalance scheduling, contributor attribution, and correlation matrices.

QUANT
⚡

Real-Time Market Data

Multi-provider WebSocket feeds from Zerodha Kite (India) and Finnhub (US). Tick-by-tick broadcast, candle aggregation at all intervals, Redis pub/sub for horizontal scaling, and distributed bar construction with provider failover.

DATA
⚠

Alerts & Breakout Scanner

Price, technical, and momentum-based alerts with WebSocket push notifications. Breakout scanner with confidence scoring, trigger/invalidation levels, and deduplication. Alert history timeline and real-time desktop notifications.

ALERTS
☁

Cross-Asset & Macro Coverage

Commodities, forex with central bank monitor, cryptocurrency workspace, ETF analytics with overlap analysis, mutual fund research with SIP calculator, bonds, US Treasury yield curve with inversion detection, and economic calendar.

MULTI-ASSET
↺

Sector Rotation & RRG

Interactive Relative Rotation Graph with 12-week trailing momentum paths. Benchmark-relative sector analysis, visual momentum/correlation display, and quadrant classification for leading, lagging, improving, and weakening sectors.

ANALYSIS
✉

News & Sentiment Analysis

Real-time and historical news ingestion with bullish/bearish/neutral sentiment classification and confidence scores. Ticker-specific and market-wide feeds with multi-period filtering and sentiment trend visualization.

RESEARCH
☑

OMS, Compliance & Governance

Order management with restricted list enforcement, immutable audit log, and compliance dashboard. Model governance with version tracking, promotion workflows, and risk limit monitoring. Ops cockpit with feed health and kill switches.

OPS
⚙

Extensible Plugin System

Build custom analysis modules with the plugin API. Included examples: RSI Divergence Scanner, Sector Rotation Monitor, and Unusual Volume Detector. Dynamic registration with full market data and chart context access.

PLUGINS
✎

Python Scripting & OpenScript

Sandboxed Python execution with security-hardened imports. Custom indicator computation and strategy scripting. OpenScript language for chart-based indicator development with script library and sharing.

Download & Install

Get started in under 5 minutes with Docker, or build from source for full development flexibility.

🗃

Docker Image

Fastest path to a running instance. Includes backend, frontend, and SQLite by default. Optional Redis and PostgreSQL via Docker Compose profiles.

$ git clone https://github.com/Hitheshkaranth/OpenTerminalUI.git
$ cd OpenTerminalUI
$ docker compose up --build
Clone from GitHub

App available at http://localhost:8000 after build

💻

Build from Source

For development and contribution. Run backend and frontend separately with hot-reload for instant feedback during development.

# Backend
$ python3.11 -m venv .venv
$ source .venv/bin/activate
$ pip install -r backend/requirements.txt
$ PYTHONPATH=. uvicorn backend.main:app --reload

# Frontend
$ cd frontend && npm ci && npm run dev

Docker Compose Profiles

# Default: Backend + Frontend (SQLite)
$ docker compose up --build

# With PostgreSQL for production workloads
$ docker compose --profile postgres up --build

# Environment variables (create .env from .env.example)
$ cp .env.example .env
$ docker compose up --build

Quick Start Guide

From clone to dashboard in three steps.

1

Clone & Configure

Clone the repo, copy .env.example to .env, and optionally add your market data API keys for full provider access.

2

Launch

Run docker compose up --build and wait for the health check to pass. The full stack (backend + frontend) starts on port 8000.

3

Explore

Open http://localhost:8000, register an account, and use Ctrl+K to open the command palette. Type any ticker symbol to begin.

Contributing

OpenTerminalUI is MIT-licensed and welcomes contributions from the community.

Development Workflow

Fork the repo, create a feature branch (feat/your-feature), write tests first, and run make gate before opening a PR. The CI checks Python compile, pytest (45% coverage threshold), frontend build, Vitest, and Playwright E2E tests.
