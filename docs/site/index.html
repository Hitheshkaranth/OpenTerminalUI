OPENTERMINALUI

A professional-grade market terminal for discretionary traders, researchers, and quant teams. Real-time charting, derivatives analytics, backtesting labs, portfolio management, and 70+ technical indicators.

Platform Capabilities
Features
Everything you need in a financial terminal — from real-time market data to institutional-grade quant workflows.
〉

Terminal Shell

Bloomberg-style GO bar (Ctrl+G), command palette (Ctrl+K) with 25+ functions, function key shortcuts (F1-F9), rolling ticker tape, and persistent workspace framing. Keyboard-first navigation with desktop and mobile layouts.

Core
⍑

Chart Workstation

Up to 9 synchronized panels with crosshair linking. 70+ technical indicators (SMA, EMA, RSI, MACD, Bollinger, VWAP, Supertrend, etc.), drawing tools with templates, replay mode, volume profile, comparison overlays. 1m to monthly timeframes with extended hours.

Charting
☰

Equity Research

8-tab security hub: overview, financials, chart, news/sentiment, ownership, estimates, peers, ESG. Fundamental metrics, earnings calendar, shareholding history, corporate actions, and peer comparison matrices.

Research
⚙

Advanced Screener

Query builder with 15+ visualization modes: tables, treemaps, heatmaps, scatter plots, radar charts, box plots, bubble charts, waterfall, RRG quadrants. Multi-market scanning across NSE, BSE, NYSE, NASDAQ with custom formulas and score-based ranking.

Screener
∇

Futures & Options

Full option chain with live Greeks (Delta, Gamma, Theta, Vega, Rho). IV analysis with term structure. Strategy builder for multi-leg positions. OI analysis, PCR tracking, IV/OI heatmaps, futures term structure, and expiry calendar.

Derivatives
★

Portfolio & Risk

Multi-portfolio management with allocation charts, benchmark overlay, correlation heatmap. Risk engine with VaR/CVaR, EWMA volatility, PCA factor exposures, stress testing. Paper trading with realistic fills, slippage, and TCA analytics.

Portfolio
⚙

Backtesting & Model Lab

10+ strategy templates. Vectorized NumPy engine with realistic execution simulation. Equity curves, 3D parameter surfaces, Monte Carlo, walk-forward analysis, monthly heatmaps. Model Lab for experiment tracking with CAGR, Sharpe, Sortino, max drawdown.

Quant
⍟

Portfolio Lab

Multi-asset portfolio backtesting with up to 200 assets. Risk parity, equal weight, momentum weighting. Strategy blends with up to 10 strategies. Rebalance scheduling, attribution analysis, and correlation matrices.

Quant
⚡

Real-Time Data

Multi-provider WebSocket feeds from Zerodha Kite (India) and Finnhub (US). Candle aggregation at all intervals. Provider waterfall with automatic failover. Multi-level caching (SQLite L1 + Redis L2). Redis pub/sub for horizontal scaling.

Data
⚠

Alerts & Scanner

Price, technical, and momentum-based alerts with WebSocket push notifications. Breakout scanner with confidence scoring, trigger/invalidation levels. Alert history timeline with deduplication and desktop notifications.

Alerts
☁

Cross-Asset & Macro

Commodities, forex with central bank monitor, crypto workspace, ETF analytics with overlap analysis, mutual funds with SIP calculator, bonds, US Treasury yield curve with inversion detection, economics calendar, and sector rotation RRG.

Multi-Asset
↺

Sector Rotation

Interactive Relative Rotation Graph with 12-week trailing momentum paths. Benchmark-relative sector analysis with quadrant classification: leading, lagging, improving, and weakening.

Analysis
✉

News & Sentiment

Real-time news ingestion with bullish/bearish/neutral sentiment classification and confidence scores. Ticker-specific and market-wide feeds with multi-period filtering and sentiment trend visualization.

Research
☑

OMS & Compliance

Order management with restricted list enforcement and immutable audit log. Model governance with version tracking and promotion workflows. Ops cockpit with feed health monitoring and kill switches.

Ops
✎

Plugins & Scripting

Extensible plugin API with included examples: RSI Divergence Scanner, Sector Rotation Monitor, Unusual Volume Detector. Sandboxed Python execution and OpenScript chart-based indicator scripting with library.

Install
Download & Deploy
Three ways to get OpenTerminalUI running.
Option 1 — Docker

Container Deploy

Fastest path. Full stack (backend + frontend + SQLite) in a single container. Optional Redis and PostgreSQL via Compose profiles.

$ git clone https://github.com/
  Hitheshkaranth/OpenTerminalUI.git
$ cd OpenTerminalUI
$ docker compose up --build
Clone Repository
Option 2 — Local Dev

Build from Source

Run backend and frontend separately with hot-reload. Best for development and contribution.

$ python3.11 -m venv .venv
$ source .venv/bin/activate
$ pip install -r backend/requirements.txt
$ PYTHONPATH=. uvicorn backend.main:app --reload

$ cd frontend && npm ci && npm run dev
View Source
Option 3 — Archive

Download Release

Download a zip of the latest main branch. Deploy to any server with Docker or run locally.

$ curl -LO https://github.com/
  Hitheshkaranth/OpenTerminalUI/
  archive/refs/heads/main.zip
$ unzip main.zip
$ cd OpenTerminalUI-main
Download .zip
Getting Started
Quick Start
From clone to dashboard in three steps.
1

Clone & Configure

Clone the repo and copy .env.example to .env. Optionally add market data API keys for full provider access.

2

Launch

Run docker compose up --build and wait for the health check. The full stack starts on port 8000.

3

Explore

Open localhost:8000, register an account, and press Ctrl+K to open the command palette. Type any ticker to begin.

Development Workflow

Fork the repo, create a branch (feat/your-feature), write tests first, and run make gate before opening a PR. CI checks: Python compile, pytest (45% coverage gate), frontend build, Vitest, and Playwright E2E.
